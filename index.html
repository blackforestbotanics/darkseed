Darkseed¶

The Pelican theme for the Botanixxx (src) blog.

installing via Pip or Pipenv:

pip install darkseed

# or

pipenv install darkseed

local development:

cd ~/projects
git clone git@github.com:blackforestbotanics/darkseed.git
cd <your blog>
pelican -t ../darkseed
